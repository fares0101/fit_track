# FitTrack Pro

تطبيق لياقة بدنية يساعدك على متابعة نشاطك اليومي، تسجيل التمارين، ورؤية تقدمك عبر رسوم بيانية واضحة.

## المزايا
- لوحة تحكم يومية تعرض الخطوات، السعرات المحروقة، الوقت النشط، والتقدم الأسبوعي.
- سجل تمارين مع إمكانية إضافة تمرين جديد (النوع، المدة، السعرات، الملاحظات).
- تقارير مرئية للتقدم: التمارين الأسبوعية، السعرات، الوزن، والدقائق النشطة.
- ملف شخصي وإعدادات (الوزن/الطول/العمر/الهدف، الوضع الداكن، وحدات KG/LBS).
- مزامنة بيانات الصحة عبر Google Fit / Apple Health (حسب الجهاز والصلاحيات).
- تخزين محلي آمن للبيانات باستخدام Hive.

## التقنيات المستخدمة
- Flutter
- Riverpod
- GoRouter
- Hive + Hive Flutter
- health + permission_handler
- fl_chart
- Lottie, Flutter Animate, Shimmer

## التشغيل محليًا
1. تأكد من تثبيت Flutter.
2. ثبّت الحزم:
   ```bash
   flutter pub get
   ```
3. شغّل التطبيق:
   ```bash
   flutter run
   ```
